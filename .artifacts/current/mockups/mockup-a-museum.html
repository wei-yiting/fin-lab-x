PE-01FIXED

Chinese query text leaked into tool arguments — “微軟最近新聞” sent verbatim to the search engine

Scene The user asks in Chinese; the agent passes the raw Chinese query to Tavily search instead of translating it to "MSFT recent news". Failures were only caught by manually reading transcripts.
Root cause A single vague prompt line — "Always think and search in English" — with no explicit rule, no example, and no guardrail at the tool-schema level.
Quantified Baseline eval: 5/8 failing → after the fix 8/8 passing (3-run stability 95.8%). This failure gave birth to the repo's entire eval framework.
